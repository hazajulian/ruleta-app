# Ruleta.app

Ruleta.app es una aplicación web de herramientas de azar, diseñada con un enfoque minimalista y premium, pensada para tomar decisiones rápidas sin distracciones ni publicidad.

## Descripción

El proyecto reúne distintas herramientas de azar en una sola plataforma, priorizando una experiencia clara, fluida y visualmente cuidada.  
Está orientado tanto a uso personal como a demostración de buenas prácticas de frontend moderno.

Herramientas incluidas:
- Ruleta
- Lanzamiento de moneda
- Dados
- Número aleatorio
- Sorteo de nombres

La aplicación es completamente responsive, con animaciones suaves y sonidos opcionales, manteniendo siempre un diseño sobrio y consistente.

## Tecnologías utilizadas

- Vite
- React
- TypeScript
- CSS Modules
- React Router
- Sonidos gestionados globalmente con persistencia en LocalStorage
